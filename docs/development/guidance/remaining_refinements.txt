## Reward-constant discrepancy fixed this pass

The NeurIPS paper's Equation 4 and appendix stated `alpha=0.1` (node
importance) and `gamma=0.05` (diversity). Re-derived directly from the code
path exercised by `jobs/curc_ace_seed.sh` (which passes no reward-weight
flags, i.e. uses argparse defaults): there is no separate alpha multiplier --
`cov_bonus`=60.0 is the node-importance term's entire scale constant -- and
`diversity_reward_weight`=0.3, not 0.05. Corrected in
`paper/iclr_ace_2027/paper.tex`, `appendix.tex`, and the hyperparameter table.
The already-submitted NeurIPS version is left unedited.
