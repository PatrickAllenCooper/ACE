================================================================================
WHAT THESE JOBS DO
================================================================================

Statistical Tests Job (1 job, 30 min):
- Confirms p<0.001 for all ACE vs baseline comparisons
- Generates effect sizes (Cohen's d)
- Creates LaTeX table for supplement
- CPU only, no GPU

Ablation Jobs (12 jobs, 2-4 hrs total):
- Tests ACE with each component removed
- Proves each component's value
- Runs in parallel (fast)
- Each job needs GPU

Total: 13 jobs, completes in 2-4 hours
