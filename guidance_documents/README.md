## Document Structure Philosophy

- **Root README.md:** Public-facing repository overview
- **guidance_documents/:** Development team guidance (this directory)
- **docs/:** Technical documentation and submission materials
- **FINAL_PAPER_RESULTS_SUMMARY.md:** Complete experimental results (root level for visibility)
- **SUBMISSION_CHECKLIST.md:** Final submission preparation (root level for visibility)

This structure separates public-facing docs, internal guidance, and technical archives.
