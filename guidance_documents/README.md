# ACE Project Documentation

This directory contains all project guidance and documentation.

## Active Documents

### guidance_doc.txt
**Main project documentation**
- Complete technical reference
- HPC workflow procedures
- Project organization and structure
- Current experimental status
- Historical changelog of improvements
- Test coverage details
- Success criteria and debugging guides

**Use for:** Understanding project architecture, running experiments, troubleshooting

### remaining_refinements.txt
**Paper improvement roadmap**
- Assessment of paper strength (ACCEPT vs STRONG ACCEPT)
- Remaining weaknesses and how to address them
- Prioritized refinement plan (Tier 1/2/3)
- Timeline estimates
- Writing mitigations if experiments not feasible
- Decision framework based on deadline

**Use for:** Planning next steps to strengthen paper before submission
