% Shared color palette for ACE paper figures.

% Conventions (semantic mapping used consistently across all figures):
%   slate   — neutral / passive entities (State, intermediate SCM nodes)
%   leaf    — agent / policy
%   rose    — proposals / candidates
%   amber   — active reasoning / lookahead
%   gold    — winner / "best" selection
%   ruby    — loser / "worst" selection / challenge nodes (colliders)
%   sky     — environment / data source / SCM root nodes
%   teal    — student model (learner)
%   vio     — learning / DPO update
%   muted   — subtle accents and dashed feedback annotations
%
% Each color has two variants: the light variant (XXX) for fills and the
% dark variant (XXXDk) for borders, text, and accent strokes.

% Usage: \input{figs/ace_palette} in the document preamble; safe to load
% multiple times (xcolor's \definecolor silently redefines).
% =============================================================================
